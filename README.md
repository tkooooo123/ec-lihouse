## 安裝流程
1. 開啟終端機，執行以下指令：
 `$ git clone https://github.com/tkooooo123/ec-lihouse.git`
2. 進入專案資料夾:
 `$ cd ec-lihouse`
3. 使用npm安裝套件
 `$ npm install`
4. 新增.env檔案，設定環境變數
```
REACT_APP_API_URL=https://ec-course-api.hexschool.io/
REACT_APP_API_PATH=sy-hex
```
5. 執行專案
 `npm install`
